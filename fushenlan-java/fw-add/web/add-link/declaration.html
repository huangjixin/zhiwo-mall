招募代理制保险营业员重要事项告知书

为确保本公司代理制保险营业员招募行为规范、透明，确保应募者的知情权，本公司向您告知如下重要事项：

  • 一、根据中国保监会规定，保险公司需对从业人员进行甄选，加强专业培训，因此代理制保险营销员必须参加本公司的专项培训并通过培训考试。
  • 二、本公司与完成培训并通过考试的人员签订个人保险代理合同（非劳动合同）并颁发执业证书，合同当事人一方为本公司，另一方为代理制保险营业员。代理制保险营业员须亮证（执业证书）营销。
  • 三、代理制保险营业员的薪酬与业绩直接挂钩，其收入形式通常为佣金（非工资）。
